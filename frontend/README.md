# Six Thinking Hats 프론트엔드

이 프로젝트는 React + Material UI 기반의 모바일 친화적 웹앱입니다.

## 개발 및 실행 방법

1. 의존성 설치
```
npm install
```

2. 개발 서버 실행
```
npm start
```

- 기본 포트: 3000 (http://localhost:3000)
- 백엔드 API 주소(`/api/solve`)는 프록시 설정을 통해 연결 필요

## 주요 파일
- `src/App.js` : 메인 UI 및 폼
- `src/index.js` : 앱 진입점
- `public/index.html` : HTML 템플릿
